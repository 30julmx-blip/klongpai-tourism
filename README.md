# 🌿 เว็บท่องเที่ยวเทศบาลคลองไผ่

ระบบสารสนเทศการท่องเที่ยวออนไลน์แบบ Interactive สำหรับเทศบาลคลองไผ่

## ✨ ฟีเจอร์

- 🗺️ **แผนที่เชิงโต้ตอบ** - แสดงหมู่บ้านและสถานที่ท่องเที่ยว
- 🎉 **กิจกรรม** - ปฏิทินกิจกรรมรายเดือน
- 🛍️ **สินค้าจากชุมชน** - แสดงผลิตภัณฑ์ท้องถิ่นแยกหมวดหมู่
- 🍜 **ร้านอาหาร** - รวบรวมข้อมูลร้านอาหารทั้งหมด
- 🏨 **ที่พัก** - ข้อมูลที่พักตามประเภท
- 🚌 **ตารางขนส่ง** - ข้อมูลขนส่งสาธารณะ
- 🎒 **แพ็คเกจแนะนำ** - ทัวร์ท่องเที่ยวแบบชุด
- 📱 **Responsive Design** - ใช้ได้บนอุปกรณ์ทุกขนาด
